MetalInfo Supplier Parser

Автоматизированный парсер справочника поставщиков сайта MetalInfo, предназначенный для сбора и структурирования информации о компаниях металлургической отрасли.

Возможности
Сбор информации обо всех поставщиках из справочника MetalInfo
Автоматический обход страниц каталога
Сохранение результатов в Excel-файл
Автоматическое обновление данных каждые 24 часа
Поддержка ручного и автоматического режимов запуска
Собираемые данные

Для каждой компании парсер извлекает:

Дату последнего обновления карточки

Как работает парсер
Открывает сайт MetalInfo.
Переходит в справочник поставщиков.
Последовательно обходит карточки компаний.
Извлекает необходимые данные.
Сохраняет результаты в Excel-файл.
Структура проекта
Ручная версия

Позволяет запустить сбор данных по требованию.

Используется для:

тестирования;
разового обновления базы;
проверки корректности работы парсера.
Автоматическая версия

Предназначена для регулярного обновления данных.

Особенности:

автоматический запуск каждые 24 часа;
обновление базы без участия пользователя;
непрерывная актуализация данных.
Запуск проекта

Установка зависимостей:

pip install -r requirements.txt

Запуск ручной версии:

parsmanual.py

Запуск автоматической версии:

parsauto.py
Результат

После завершения работы формируется Excel-файл с полной информацией о поставщиках из справочника.

Полученные данные могут использоваться для:

формирования клиентской базы;
анализа рынка;
поиска потенциальных партнеров;
импорта в CRM-системы;
подготовки аналитических отчетов.
Особенности реализации

В процессе разработки были решены следующие задачи:

автоматизированный обход многостраничного каталога;
извлечение структурированных данных из HTML-разметки;
обработка отсутствующих данных;
сохранение данных в Excel-формат;
автоматизация регулярного обновления информации.
Ограничения

На данный момент проект запускается непосредственно через Python.

Упаковка в исполняемый файл (.exe) не реализована, поэтому для работы требуется установленный Python и необходимые зависимости.

В репозитории приложен файл `metalinfo_contacts.xlsx`, содержащий пример данных, полученных в результате работы парсера.

Автор

Проект разработан в рамках практического изучения автоматизации веб-сбора данных и работы с Python.
